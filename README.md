# amqplib-flow
## amqp.node type declarations for Flow

Only promise-based API is supported

## Installation

Install via npm:

```sh
npm install --save amqplib-flow
```

Then add a line to `[libs]` section of yout `.flowconfig`:

```
[libs]
...
node_modules/amqplib-flow/types.js
...
```
